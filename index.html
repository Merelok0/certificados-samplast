Certificado de Calidad

Llena el formulario y genera el PDF. Usa 📷 para leer la etiqueta con IA automáticamente.

Ej. 18*20*2.0 → 18" ancho, 20 µm, 2.0 kg. Pre-estirado siempre usa 18" y en ancho muestra 420*.

Medidas y valor medido del certificado se calculan desde estos datos (mm = pulgadas × 25,4).

El formato etiqueta arma el nombre STRETCH FILM, las celdas MEDIDAS/VALOR MEDIDO y el texto de apariencia si hay color. La fecha de emisión se completa con +1 día si la dejas vacía. Use ABRIR_CERTIFICADO.bat y luego Guardar PDF… para elegir carpeta y nombre (como pestaña Excel).
